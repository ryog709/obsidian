# Miraitect 株式会社-Web サイト-HTML-CSS コード

## 概要

Miraitect 株式会社の Web サイト制作に関する HTML/CSS コードのメモ。ヘッダーナビゲーション、サービスリスト、マウスストーカーボタンなどの実装コードが含まれています。

## 内容

### ヘッダーナビゲーション

- PC 用ナビゲーション（7 つのメニュー項目）
- SP 用ハンバーガーメニュー
- ロゴ画像の配置

### サービスリストセクション

12 種類のサービスカテゴリー：

**築く（build）カテゴリー:**

1. 瓦屋根
2. 外壁
3. 板金
4. 塗装
5. 防水
6. エクステリア

**造る（create）カテゴリー:** 7. 大工 8. 左官 9. 軽天

**支える（support）カテゴリー:** 10. 住宅設備 11. 断熱施工 12. 太陽光

各サービスには画像、タイトル、説明文が含まれています。

### About Skill セクション

- パララックス効果を使用したセクション
- 「手に職を持つこと」というメッセージ
- 背景画像のパララックスアニメーション

### マウスストーカーボタン

- 固定位置のマウスストーカー要素
- PC 表示のみ（1099px 以下では非表示）
- ホバー時のアニメーション効果
- 背景のぼかし効果（backdrop-filter）
- 矢印アイコンのマスク画像使用

## コードの特徴

- BEM 記法を使用したクラス命名
- レスポンシブ対応（PC/SP）
- JavaScript 連携（js-プレフィックス付きクラス）
- 画像の遅延読み込み（loading="lazy"）
- アクセシビリティ対応（alt 属性）
