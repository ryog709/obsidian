# ヘッダーナビゲーション-レスポンシブ対応

## 概要
PC用ナビゲーションとSP用ハンバーガーメニューを含むレスポンシブ対応のヘッダーコンポーネント。

## 特徴
- BEM記法を使用
- PC/SP両対応
- JavaScript連携（`js-hamburger`, `js-sp-nav`）
- アクセシビリティ対応（適切なalt属性）
